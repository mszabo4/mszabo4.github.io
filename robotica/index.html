Projecte de Robòtica

A continuacio explicaré el meu projecte de robotica, que consisteix en obrir una llum amb la veu ,pero com he tingut diferents problemes explicaré primer totes les eines compatibles amb el meu microcontrolador , els problemes que he tingut de configuració i reconeixement del microcontrolador, els circuits provats en dibuixos tinkercad i les solucions trobades.

Eines

A continuació veuràs un llistat amb diferents eines que utilitzarem durant el curs i que permetran solucionar problemes:

  • Arduino IDE (entorn de desenvolupament per plaques compatibles amb arduino), es un llenguatge derivat de C++ i processing
  • Processing amb arduino library que ens permet utilitzar totes les biblioteques de processing :exemples figures 3D, Impressio 3D, Realitat virtual i Realitat augmentada, inteligencia artificial, Reconeixement d'objectes i cares , video, etc.... Objectiu del tercer trimestre. L'únic problema és que hem de tenir connectat l'ordenador a l'arduino si hem d'utilitzar el microfon o la camara o els altaveus o el teclat de l'ordinador,
  • Mu editor és un editor de codi python molt sencill d'utilitzar i compatible amb microcontroladors. Es pot descarregar des de la seva web. Hi ha diferents modes de treball amb Mu: "Python3" per escriure codi python i instalar biblioteque externes, "Pygame Zero" és una bliblioteca python per a fer jocs però més senzilla que pygame, "BBC microbit" per utilitzar la placa microcontroladora anglesa microbit, "Circuitpython" és un llenguatge de programacio derivat de python per plaques adafruit
  • Diferents plaques microcontroladores: Arduino, Esp32-S3, Xiao Samd21 , Adafruit Circuit Playground Express.
